Создайте имитацию "мини-интернет магазина"
В главном окне отображается список всех товаров 
(каждый товар, это отдельный элемент управления, созданный вами вручную)
Для отображения товаров используйте ListBox с шаблоном панели WrapPanel
Реализуйте возможности:
- Добавление нового товара
- Удаление выбранного товара
- Детальный просмотр выбранного товара (можно в отдельном окне)
- Редактирование выбранного товара

Каждый товар включает в себя:
- название
- изображение
- цена
- рейтинг


- категория
- описание
- автор (обычная строка)
- дата добавления
